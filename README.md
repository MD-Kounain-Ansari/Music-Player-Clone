# 🎵 Spotify Clone

A Spotify-inspired music player built using **HTML, CSS, and Vanilla JavaScript**. This project allows users to play songs, control playback, adjust volume, and browse a playlist through a clean and responsive interface.

## 🚀 Live Demo

** Click Here to See the Website **
👉  https://kounain-music-player.netlify.app/

---

## ✨ Features

- 🎵 Play and Pause songs
- ⏮ Previous & Next controls
- 📃 Playlist with multiple songs
- 🎼 Display current song title and artist
- ⏱ Interactive seek bar
- 🔊 Volume control
- 🔇 Mute / Unmute functionality
- 📱 Responsive design for different screen sizes

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
